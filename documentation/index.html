Understanding

This is a Login form that use CSS3 and jQuery Tricks to look like windows 8 login screen.

 

Inside of the project, you will found a font called: ArchitectsDaughter.ttf, You can only use it in this project. if you want to extend it in other projects contact  kimberlygeswein@gmail.com

 

You can modify, edit, change, whatever you want.

 

I'll explain every file in this documentation. So you can modify and make it work like you want.

 

Installation

Just Copy all the file and folder structure inside your website.

 

To change the appearance of the login, open "js" folder and edit "MetroLogin.js", you will find a couple of options.

 

I use jquery 1.8.3, but you can use 1.3 or later version.

 

Read the next topic "Configurations"

 

Configurations

There are some configurations that I think you may like. And all of them you can find it in: JS (folder) >> MetroLogin.js

 

  

var bgColor        = "#014051";
bgColor: This opcion change the background and button color. Just change it and you are ready to go.
 
 
var LockWallpaper  = "img/back2.jpg";
LockWallpaper: This option change the lockscreen picture. Just set a good looking image and see the result. The css will take the image and adapt it to a 100% of the user screen.

WeekDays: If you want to change the lenguage, is so easy, just change the values and that's it.

Months: This array, change the names of the months. Just change this values.
 
 
var MonthSeparator = "of";
MonthSeparator: This is the text that separate the day from the month... example: "21 of October", if you change it to "de", it will appear "21 de Octuber"
 
 
var UserEnterMessage = "Please, enter your username.";
UserEnterMessage: This is the enter message, after the user click the image, this message appear.

LoginMessage: This is the message that appear when the user type a valid or invalid user and password.
 
 
var ErrorMessage     = "Username and password are not correct.";
ErrorMessage: Change the message when the user type a wrong user and password.

{

$Valid = 1;

}

As you can see, this function most return 1 if the user and password are correct, and most return 0 if the user and/or password .

 
Just change what ever you want to make this work like you want.
 
This rutine is called via ajax (line 352 of MetroLogin.js)... you do not need to modify it... 
 
 
There are five variables:
 
$UserName: This is the user value from the text field.
 
$Password: This is the user password.
 
$chkRemember: 1 if the user check the "Remember Me" check box. or 0 if the user dont check it.
 
$CurrentColor: The configuration of the color picked or selected from the user
 
$CurrentBack: The name with extension of the current wallpaper
 

IMPORTANT:

There are a invisible field called "HiddenPass", this input save the password that the user type... this field helps me to create the illution of the "eye" to see the pass when the user clicks and hold that image. So, the real password are in this field.

Line 38 and 53. Use those functions to set any kind of color or background.

Redirect to Any Page!

After the PHP function is called via AJAX. You can mannage what do you want to do.

 

Check "MetroLogin.js" file again. Line 21.

 

There is a super simple function:

  

function isCorrect(IsCorrect)
{

if(IsCorrect == 1)

{

window.location.replace("welcome.html");

}

else

{

UnLoading();

}

}
 

Simple, the parameter value "IsCorrect", contains the returned value from the "loginCheck.php" file. 

 

So, you can modify this function if you want to do some special thing. As you can see, if the returned value is "1", I redirect the user to a page called "Welcome.html".

 

UnLoading(): Do not change this function unless you have a fully understanding what you are doing. This funtion call a couple of other functions that hide the login, appear the login error message and wait fo the user press the "Accept" button to show the inputs again.
